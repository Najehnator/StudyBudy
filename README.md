## 2. Tekniker som används

Projektet använder följande tekniker:

- Python
- Flask
- HTML
- CSS
- PostgreSQL
- psycopg2
- Werkzeug för lösenordshashning

---

## 3. Krav för att kunna köra projektet

För att kunna köra projektet behöver ni ha följande installerat:

- Python 3
- pip
- PostgreSQL eller åtkomst till PostgreSQL-server
- Python-biblioteken:
  - Flask
  - psycopg2-binary
  - Werkzeug

Installera biblioteken med:

pip install Flask psycopg2-binary Werkzeug
